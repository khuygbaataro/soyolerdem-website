## 🚀 Хэрхэн ашиглах

### Локал орчинд үзэх

1. Бүх файлыг задалсан хавтас руу орно уу
2. `index.html`-ийг аль нэг орчин үеийн хөтчөөр нээнэ (Chrome, Safari, Firefox, Edge)

### Сервер дээр байрлуулах

Энэ бол **100% статик сайт** — ямар нэг сервер хэрэггүй:

- **GitHub Pages** (үнэгүй) — `soyolerdem-website/` хавтсыг GitHub repo болгож `gh-pages`-д push хийх
- **Netlify / Vercel** (үнэгүй) — хавтсыг drag-and-drop хийх
- **Өөрийн сервер** — FTP-ээр public_html хавтсанд хуулах
- **AWS S3** — static website hosting идэвхжүүлж хуулах

## ✏️ Агуулгыг яаж өөрчлөх

### 1. Текст өөрчлөх
Хэрэгтэй `.html` файлаа ямар нэгэн кодын редактор (VS Code, Sublime) эсвэл notepad-аар нээж, текстээ шууд солино.

### 2. Зураг солих
Одоогоор бүх зураг **Unsplash** placeholder ашиглаж байна. Бодит зургаа ашиглахдаа:

**Алхам 1:** Зургуудаа `assets/images/` хавтсанд хадгалах:
```
assets/images/
├── hero-students.jpg
├── chairman-dorjdagva.jpg
├── shilin-camp.jpg
├── news-1.jpg
└── ...
```

**Алхам 2:** HTML файл дотор URL-г солих:
```html
<!-- ХУУЧИН -->
<div style="background: url('https://images.unsplash.com/photo-...');">

<!-- ШИНЭ -->
<div style="background: url('assets/images/hero-students.jpg');">
```

> **ТАЙЛБАР:** `jp/index.html` хуудас дотор бол `../assets/images/...`, pages/ доторх бол `../assets/images/...` замаар хандана (нэг түвшин дээш).

### 3. Өнгө өөрчлөх
`assets/css/main.css` файлын эхний хэсэгт байгаа `:root { --ink: ... }` блок дахь өнгөнүүдийг солих. Солисон даруйд бүх 19 хуудаст автоматаар хамаарна.

### 4. Шинэ хуудас нэмэх
1. `pages/` дотор ямар нэг хуудсыг хуулж шинэ нэрээр тавина (жишээ нь `pages/about.html` → `pages/shinii-huudas.html`)
2. Гарчиг, breadcrumb, контентоо засна
3. Nav цэс рүү шинэ линк нэмэхдээ **бүх 19 файл** дотрох `<ul class="nav-menu">` блокыг шинэчилнэ (эсвэл JavaScript-ээр шингээх)
