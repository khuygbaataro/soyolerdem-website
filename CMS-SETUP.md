## Юу хийх гэж байна?

Одоо энэ вебсайт нь **Decap CMS** (хуучин нэр нь Netlify CMS) -тэй холбогдсон. Энэ нь сургуулийн ажилтанд:

- ✅ WordPress шиг энгийн админ хэсэгтэй
- ✅ Мэдээ нэмэх, засах, устгах
- ✅ Сонингийн PDF байршуулах
- ✅ Төгсөгчдийн мэдээлэл оруулах
- ✅ Эрдэм шинжилгээний бүтээлийн жагсаалт бүрдүүлэх
- ✅ Ажлын байр, элсэлтийн зарлах

Бүгдийг **кодын мэдлэггүйгээр** хийх боломжтой.

## 🔧 Суулгах — Техникийн тал

### Алхам 1: GitHub repo үүсгэх

1. [github.com](https://github.com)-д нэвтрэнэ үү
2. **New repository** дарна
3. Нэр: `soyolerdem-website`
4. **Public** сонгоно (үнэгүй байх шаардлага)
5. Энэ бүх файлыг upload хийнэ

### Алхам 2: Netlify дээр хостлох

1. [netlify.com](https://netlify.com) дээр бүртгүүлнэ (GitHub-аар нэвтрэх нь хялбар)
2. **Add new site → Import an existing project**
3. GitHub-ийг сонгоод `soyolerdem-website` repo-г сонгоно
4. Deploy дарна — **3-5 минут дотор сайт онлайн болно**
5. URL: `xxxxx.netlify.app` гэсэн хаяг өгөгдөнө. Үүнийг `soyolerdem.edu.mn` хаягтайгаа солих боломжтой (Netlify-ийн Domain settings-ээс)

### Алхам 3: Netlify Identity идэвхжүүлэх (нэвтрэлт)

1. Netlify админ хэсэгт ороод **Site settings → Identity**
2. **Enable Identity** дарна
3. **Registration preferences** → **Invite only** сонгоно (хэн ч бүртгүүлэхгүй байхын тулд)
4. **External providers** → GitHub нэмэх (сонголттой, илүү хялбар нэвтрэлт)

### Алхам 4: Git Gateway идэвхжүүлэх

1. Identity settings хэсэгт **Services → Git Gateway → Enable**
2. Энэ нь CMS-ийг GitHub-тай холбоно

### Алхам 5: Ажилтнуудыг урих

1. **Identity** tab → **Invite users**
2. Ажилтны и-мэйлийг оруулна
3. Тэд и-мэйлийн линкээр дамжуулан нэвтэрнэ

## 👤 Сургуулийн ажилтан хэрхэн ашиглах

### Нэвтрэх

1. `soyolerdem.edu.mn/admin` хаяг руу орно
2. И-мэйл, нууц үгээ оруулна (эхний удаа и-мэйлээр ирсэн тохиргооны линкийг дарна)
3. Нэвтэрсний дараа WordPress шиг хэрэглэгчийн интерфейс нээгдэнэ

### Удирдах цэс

Зүүн гар талын цэсэнд 5 төрлийн контент харагдана:

- 📰 **Мэдээ мэдээлэл** — сайтын мэдээ
- 📖 **Сонин хэвлэл** — PDF дугаарууд
- 🎓 **Төгсөгчид** — карьерын жагсаалт
- 🔬 **Эрдэм шинжилгээ** — CITI Science бүтээл
- 💼 **Ажлын байр · Элсэлт** — зарлалууд

### Мэдээ нэмэх

1. **📰 Мэдээ мэдээлэл** → **New Мэдээ** дарна
2. Талбаруудыг бөглөнө:
   - **Гарчиг** — зайлшгүй
   - **Огноо** — автоматаар өнөөдрийн огноо
   - **Ангилал** — жагсаалтаас сонгох (Элсэлт, Үйл явдал, Амжилт...)
   - **Онцолсон мэдээ** — нүүр хуудасны гол мэдээ болгох бол ☑
   - **Нүүр зураг** — drag & drop эсвэл browse
   - **Богино тайлбар** — 2-3 өгүүлбэр
   - **Бүтэн нийтлэл** — bold, italic, list, зурагтай рич текст редактор
3. **Save** дарахад draft болж хадгалагдана
4. **Publish** дарахад сайтан дээр гарна (1-2 минут дотор)

### Preview хийх

Хадгалахаас өмнө баруун талын **Preview** tab-аар тухайн хуудас сайтан дээр хэрхэн харагдахыг үзэх боломжтой.

### Засах, устгах

- **Мэдээ мэдээлэл** → жагсаалтаас тохирох мэдээ дээр дарна
- Засах: Талбаруудыг шинэчлэнэ → **Save**
- Устгах: Дээд талын **Delete** товч

## 🚨 Алдаа засах

### Асуудал: "Админд нэвтрэх боломжгүй"

- И-мэйл, нууц үгээ шалгана
- Сайн байхгүй бол Netlify админаас "Invite user" дахин илгээнэ
- Browser-ийн cookie, cache цэвэрлэнэ

### Асуудал: "Мэдээ хадгалагдахгүй байна"

- Интернэт холболтоо шалгана
- "Save" товч 30 секунд хүртэл унших хугацаа хэрэгтэй
- Alert гарвал screenshot авч шийдэл хайна

### Асуудал: "Зураг хуулагдахгүй байна"

- Зургийн хэмжээ 10MB-аас бага байх
- Формат: JPG, PNG, WebP (GIF дэмждэггүй)
- Файлын нэрэнд зай, тусгай тэмдэгт бүү оруул

### Асуудал: "Шинэ мэдээ сайтан дээр гарахгүй байна"

- 1-2 минут хүлээгээрэй (GitHub-аас автоматаар pull хийгдэнэ)
- Browser-ийг F5 эсвэл Ctrl+Shift+R дарж сэргээх
- "featured" checkbox идэвхжсэн эсэхийг шалгах

## ✅ Бэлэн болсны дараа

Нэг эрдэм шинжилгээний багш, нэг боловсролын ажилтан CMS-ийг ашиглаж эхлэх ёстой. Эхний 2 долоо хоногт:

1. ✓ 5-10 хуучин мэдээ оруулах (одоогийн medee.html дахь static мэдээг нүүлгэн)
2. ✓ Одоо байгаа 47, 46, 45 сонин дугааруудыг PDF-ээр upload хийх
3. ✓ 20-30 алдартай төгсөгчдийн мэдээлэл оруулах
4. ✓ Бүх CITI Science бүтээлийг сэтгүүл болгон бүртгэх

Ингэснээр сайт динамик, хөдөлгөөнтэй болж, сургуулийн ажилтнууд хүссэн үедээ шинэчлэх боломжтой болно.
