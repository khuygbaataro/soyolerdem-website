# Эрдэм шинжилгээний бичиг (боть) · CMS collection

Энэ хавтас нь админаас upload хийсэн боть сэтгүүлийн metadata-г хадгалдаг.

## Бүтэц

```
content/research-journals/
├── README.md                     — энэ файл
├── 2025-1.md                     — 1(01) 2025 боть
├── 2024-1.md                     — 1(01) 2024 боть
└── pdfs/
    ├── erdem-2025-1.pdf          — PDF файл өөрөө
    └── erdem-2024-1.pdf
```

## Шинэ боть нэмэхэд

1. `/admin/` рүү орж, **📚 Эрдэм шинжилгээний бичиг (боть)** collection сонгох
2. **New боть** дарна
3. Гарчиг, дугаар, он, PDF файлаа upload хийнэ
4. Save → Published
5. Сайтын `/pages/erdem.html`-д автоматаар карт хэлбэрээр харагдана

## PDF ажиллах зарчим

Карт дээр дарсан үед `/assets/js/pdf-viewer.js` нь PDF.js-ээр файлыг ачаалж, ном хэлбэрээр (2 хуудас зэрэгцээ) харуулна. Зүүн-баруун сумнуудаар хуудас эргүүлнэ.
