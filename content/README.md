# Content хавтас

Энд бүх CMS-ийн контент хадгалагдана. Админ хэсгээс өөрчлөлт хийхэд файлууд автоматаар энд нэмэгдэж, өөрчлөгдөнө.

## Бүтэц

```
content/
├── medee/           📰 Мэдээ мэдээлэл (.md файлууд)
├── sonin/           📖 Сонин хэвлэлийн дугаарууд
├── tugsugchid/      🎓 Төгсөгчдийн мэдээлэл
├── erdem/           🔬 Эрдэм шинжилгээний бүтээлүүд
├── ajlyn-bair/      💼 Ажлын байр, элсэлтийн зарлалууд
└── images/          🖼️ CMS-ээр оруулсан зургууд
```

## Файлын формат

Бүх контент **Markdown (.md)** хэлбэрээр хадгалагдана. Файлын дээд хэсэг front matter (YAML metadata) агуулна:

```markdown
---
title: "Гарчиг"
date: "2026-04-15"
category: "elselt"
---

Бүтэн текст энд...
```

## Гараар файл засах

Техник мэдлэгтэй бол admin UI-аас биш GitHub-аас шууд ч засаж болно. Харин тохиргоо буруу болвол CMS ажиллахгүй болохоор **админ UI-аас засах нь зөв**.
